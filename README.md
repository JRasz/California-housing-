California Housing Price Prediction

This project predicts housing prices in California using key features such as location (longitude, latitude), housing age, number of rooms, number of bedrooms, population, households, median income, and ocean proximity. The model is built using Linear Regression, and the project includes several visualizations that explore relationships between these features.

Libraries Used:

-Scikit-learn

-Matplotlib

-Seaborn

The repository includes graphs and exploratory analysis to show how different factors affect housing prices.
